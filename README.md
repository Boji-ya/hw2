支出紀錄工具（Expense Tracking Tool）

本專案為軟體工程課程作業，使用 Python 實作簡單的支出紀錄與視覺化功能。

執行環境

Python 3.x

需安裝 matplotlib 套件

執行方式
1. 輸入支出資料

請先執行輸入模組，新增支出紀錄並產生 data.txt 檔案。

python input.py

2. 顯示支出視覺化結果

請確認 data.txt 與 visualize.py 位於同一資料夾中，
在終端機中執行以下指令：

python visualize.py


執行後會顯示依支出類別統計的圓餅圖。

備註

若尚未安裝 matplotlib，請先執行：

pip install matplotlib
